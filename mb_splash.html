About MetaBuild
  • MetaBuild is a distributed C++ package manager and project generator written in portable Python 3.8+ code.
  • Python-based syntax is used for the MetaBuild files. The builder commands are as close as possible to Buck from Facebook.
  • Compile all of the C++ code from source and allow sharing external C++ dependencies without duplication.
  • Unlike Buck, MetaBuild doesn't actually drive the compilation of the C++ code:
    • Generates native Xcode and Visual Studio projects.
    • Generates CMake files for Android and Linux.
  • Supported external dependencies:
    • download via HTTP/S
    • download from Artifactory
    • download code straight out of GIT
  • MetaBuild is fast:
    • cache downloads from remote
    • cache extracted dependencies
    • allow shallow git clones of external repos with lots of history
    • allow local linking of external dependencies for local development
    • Python Requirements
  • 2021-08 - 2023-01 : >= ver. 3.8
  • 2023-01 - ... : >= ver. 3.10
